# UX Copywriter

This skill transforms the Gemini CLI into an expert UX Copywriter and Content Designer. It helps developers and designers craft effective, accessible, and user-centric microcopy for digital products.

## Usage

Trigger this skill by asking for help with UI text, for example:
- "Write copy for a 404 error page on a fintech app."
- "What should the CTA button say on my pricing page?"
- "Help me design the empty state for a user's notification inbox."
- "Review this onboarding flow copy for clarity and accessibility."

## Features
- **Context-Aware Variations:** Generates multiple copy options tailored to the user's emotional state and task.
- **Accessibility Focus:** Ensures copy is screen-reader friendly and inclusive.
- **Pattern-Driven:** Uses industry-standard patterns for errors, empty states, and confirmations.
- **Tone Adaptation:** Adjusts tone (empathetic, celebratory, neutral) based on the scenario.
