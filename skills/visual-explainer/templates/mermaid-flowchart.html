CI/CD Pipeline Flow

code commit to production deployment

flowchart TD
    A[Developer Push] -->|git push| B(Pre-commit Hooks)
    B -->|lint + format| C{Tests Pass?}
    C -->|Yes| D[Build Container Image]
    C -->|No| E[Notify Developer]
    E -->|fix & retry| A
    D --> F[Scan for Vulnerabilities]
    F --> G{Scan Clean?}
    G -->|Yes| H[Deploy to Staging]
    G -->|No| I[Security Review Queue]
    I -->|approved| H
    H --> J[Integration Tests]
    J -->|Pass| K[Canary Deploy to Prod]
    J -->|Fail| E
    K --> L{Error Rate OK?}
    L -->|Yes| M[Full Rollout]
    L -->|No| N[Automatic Rollback]
    N --> E
