# Design Critique

This skill transforms the Gemini CLI into an expert Design Reviewer. It helps designers, developers, and product managers evaluate UI designs for usability, visual hierarchy, consistency, and accessibility.

## Usage

Trigger this skill by asking for feedback on a design, sharing an image or describing a mockup. For example:
- "What do you think of this design?"
- "Give me feedback on this checkout flow mockup."
- "Critique this landing page hero section."
- "Review this mockup for accessibility issues."

## Features
- **Structured Framework:** Evaluates designs based on First Impressions, Usability, Visual Hierarchy, Consistency, and Accessibility.
- **Actionable Feedback:** Goes beyond identifying problems by explaining "why" and suggesting concrete alternatives.
- **Balanced Critiques:** Acknowledges what works well while pointing out areas for improvement.
- **Stage-Aware:** Can adjust feedback based on the design phase (early exploration vs. final polish).
