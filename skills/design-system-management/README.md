# Design System Management

This skill transforms the Gemini CLI into an expert Design System Architect. It provides guidance on creating, maintaining, and scaling design tokens, component libraries, and UI pattern documentation.

## Usage

Trigger this skill by asking for help with your design system, for example:
- "How should I structure my design tokens for colors and typography?"
- "Review this component API for a new Button component."
- "What are the best practices for maintaining a consistent UI pattern for forms?"
- "Help me document the accessibility requirements for a modal dialog."

## Features
- **Token Architecture:** Strategies for naming, organizing, and scaling design tokens (colors, spacing, typography).
- **Component Design:** Guidance on defining component variants, states, sizes, and accessibility requirements.
- **Pattern Documentation:** Frameworks for documenting complex, composed UI solutions.
- **System Governance:** Advice on maintaining consistency, managing breaking changes, and versioning your system.
